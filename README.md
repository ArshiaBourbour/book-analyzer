# 📚 Book Market AnalyzerA simple Python project that collects the **Top 10 best-selling books** from three popular Iranian online bookstores and saves the results as CSV files.This project was built as a learning exercise to practice **Web Scraping**, **Data Processing**, **Docker**, and **GitHub Actions**.## Supported Websites* IranKetab* 30Book* DigiKala## Features* Scrape the top 10 best-selling books from each website.* Save the results as separate CSV files.* Generate a new CSV file every time the project runs.* Run the project using Docker. ## Tech Stack* Python* Requests* BeautifulSoup4* Pandas* Docker* GitHub Actions## Project Structure```textbook_analyzer/│├── scraper/├── data/├── main.py├── requirements.txt├── Dockerfile└── .github/```## Getting StartedClone the repository:```bashgit clone https://github.com/ArshiaBourbour/book-analyzer.gitcd book-analyzer```Create and activate a virtual environment:```bashpython -m venv venv```macOS / Linux```bashsource venv/bin/activate```Windows```bashvenv\Scripts\activate```Install the required packages:```bashpip install -r requirements.txt```Run the project:```bashpython main.py```## Running with DockerBuild the image:```bashdocker build -t book_analyzer .```Run the container:```bashdocker run --rm -v "$(pwd)/data:/app/data" book_analyzer```## OutputAfter each execution, three CSV files are created inside the `data` directory, one for each website.Example:* `iranketab_20260628_074906.csv`* `30book_20260628_074906.csv`* `digikala_20260628_074906.csv`## NotesThis project is intended for educational purposes.
